Questions fréquentes

Comment créer un compte ?

Téléchargez Epargn+, ouvrez l'application et inscrivez-vous avec votre numéro de téléphone. Un code de vérification vous est envoyé par SMS.

Comment déposer ou épargner de l'argent ?

Depuis votre espace client, choisissez un objectif d'épargne puis suivez les instructions de dépôt affichées. Contactez-nous si un dépôt n'apparaît pas.

Comment retirer mes fonds ?

Rendez-vous dans votre espace client, sélectionnez le solde à retirer et validez la demande. Le traitement se fait selon les délais indiqués dans l'application.

J'ai oublié mon code / je n'arrive pas à me connecter

Utilisez l'option de reconnexion par SMS sur l'écran de connexion. Si le problème persiste, écrivez à ceo@epargnplus.com.

Comment supprimer mon compte ?

Voir la page Suppression de compte, depuis l'application ou par email.

Mes données sont-elles protégées ?

Oui. Consultez notre Politique de confidentialité pour le détail des données collectées et conservées.
