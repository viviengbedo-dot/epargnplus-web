❓ Questions fréquentes
Comment faire un dépôt ? ›

Allez dans "Accueil" → "Déposer". Sélectionnez votre opérateur (Orange Money, MTN MoMo, Wave) et suivez les instructions. Votre dépôt sera confirmé sous 24h.

Comment fonctionne le KYC ? ›

Le KYC (vérification d'identité) est obligatoire pour accéder aux retraits. Fournissez une photo de votre CNI et un selfie. Notre équipe valide votre dossier sous 2h.

Mon dépôt tarde à être confirmé ? ›

Les dépôts sont confirmés manuellement par notre équipe dans un délai de 2 à 24h ouvrées. Si après 24h votre dépôt n'est pas confirmé, ouvrez un ticket de support.

Comment utiliser un code promo ? ›

Allez dans Paramètres → "Appliquer un code promo". Entrez votre code et le bonus sera immédiatement crédité sur votre compte.
